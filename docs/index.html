FAQ

When will dahlia be ready for use?
It's difficult to tell, but we hope to have the first public version ready before the end of March 2026

What are dahlia's goals?
Dahlia aims to blend the minimal nature of terminal-based editors (like NeoVim and Kakoune) with the familiarity of GUI editors (like VSCode or IntelliJ)

Does / will dahlia support extensions?
Eventually! We plan to add basic extension support once the actual editor is finished, and then we'll extend from there
